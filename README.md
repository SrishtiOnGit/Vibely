# Vibely
Vibely is an app which will give you a personalised music experience depending on your mood.

## Key Features:
1. Personalised Songs
2. Smart Playist
3. Mood-Based Listening
4. Save Favourites

## Tech Stack:
1.React.js
2.Soft Aurora's wave animation.
3.CSS

##File Structure:
--> I kept the components, styles and pages in the components, styles, and pages folder.

##Steps to Start:
1.You can click Start Listening Button.
2.You will land on the Login Page where you can directly click on the login button and no need to fill any details.
3.You could see our dashboard.
4. You can select your mood.
5.Click on the Generate Vibe Button.
6. After that you will have a song for generate vibe.
